## [10/12] 기술 문서 / 원페이저 링크

제출 링크: 【GitHub 레포 README 또는 Notion 공개 페이지 — 아래 내용】

**원페이저에 넣을 내용:** (레포의 `docs/superpowers/specs/2026-07-16-mulle-design.md` 6~7절 기반)

> ## 물레 MULLE — 기술 원페이저
> **아키텍처**: 백엔드 없는 순수 dApp. Next.js(wagmi/viem) → GIWA Sepolia 직접 통신
> **컨트랙트**: MockKRW(데모용 원화, 실제 KRW 스테이블로 교체 가능한 구조) / MulleFactory / Mulle(계 1개 = 인스턴스 1개)
> **상태머신**: 모집중 → 진행중 → 완주 | 파탄
> **핵심 메커니즘**:
> - 순번: 온체인 제비뽑기(blockhash 시드 Fisher–Yates) 또는 계주 제안+전원 서명 동의
> - 미납 방어: 보증금(0~2회분) 자동 슬래싱, 소진 시 자동 파탄
> - 파탄 정산: 잔여 보증금 반환 후 잔액을 미수령 멤버에게 납입액 비례 분배
> - 지급: pull 방식 claim (수령자 지갑 문제 격리)
> - 자동화: 누구나 호출 가능한 settle() + 0.1% 호출 보상 = 서버리스 회차 마감
> **알려진 한계와 계획**: blockhash 랜덤은 친목계 소액용 → 오픈계에서 commit-reveal 업그레이드 / 감사 전 메인넷 미배포 원칙
